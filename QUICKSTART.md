# 🚀 TEZKOR START - Aiogram Bot

## ⚡ 3 Daqiqada Bot Tugatish

### 1️⃣ Token Olish (2 soat)
```
Telegram → @BotFather → /newbot → Token ko'chir
```

### 2️⃣ .env Yaratish (1 daqiqa)
```bash
cp .env.example .env
nano .env
# BOT_TOKEN=XXXX... qo'shing
```

### 3️⃣ Ishga Tushirish (1 daqiqa)
```bash
python run_bot.py
```

## ✨ Bot Ishlaydi!

Avval qo'l bilan Telegram da botni qidiring va `/start` yuboring.
